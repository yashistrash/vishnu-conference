## SENSOR CONNECTIONS (All sensors follow same pattern)

Each sensor has 3 pins:

Soil Moisture Sensor:
  VCC  →  5V Power Rail (Red wire)
  GND  →  GND Power Rail (Black wire)
  AOUT →  GPIO 34 (Yellow/Signal wire)

Rain Sensor:
  VCC  →  5V Power Rail (Red wire)
  GND  →  GND Power Rail (Black wire)
  DO   →  GPIO 35 (Yellow/Signal wire)

pH Sensor:
  VCC  →  5V Power Rail (Red wire)
  GND  →  GND Power Rail (Black wire)
  PO   →  GPIO 36 (Yellow/Signal wire)

DHT22:
  VCC  →  5V Power Rail (Red wire)
  GND  →  GND Power Rail (Black wire)
  DATA →  GPIO 4 (Yellow/Signal wire)

## LED CIRCUITS (All follow same pattern)

For EACH LED:
  ESP32 GPIO → 220Ω Resistor → LED Anode (+) → GND

Specific connections:

Red LED (Pump Active):
  GPIO 26 → 220Ω → LED+ → GND

Green LED (System OK):
  GPIO 27 → 220Ω → LED+ → GND

Blue LED (Uploading):
  GPIO 25 → 220Ω → LED+ → GND

Yellow LED (Warning):
  GPIO 33 → 220Ω → LED+ → GND

CRITICAL: LED polarity matters!
  Anode (+) = Longer leg = Connects to resistor
  Cathode (-) = Shorter leg = Connects to GND

## LCD DISPLAY (I2C)

4-pin connection:

LCD Module:
  VCC → 5V Power Rail
  GND → GND Power Rail
  SDA → GPIO 21
  SCL → GPIO 22

I2C Address: Usually 0x27 (sometimes 0x3F)

═══════════════════════════════════════════════════════════════

## BREADBOARD LAYOUT

Power Rails:
  [+] ═══════════════════════  (Red stripe) = 5V
  [-] ═══════════════════════  (Blue/Black stripe) = GND

Component Placement:
  - Place ESP32 across center gap
  - LEDs in rows on one side
  - Sensors can plug directly or use wires
  - Keep wires neat and organized

═══════════════════════════════════════════════════════════════

## WIRE COLOR CODING (Recommended)

RED    = Power (5V, VCC)
BLACK  = Ground (GND)
YELLOW = Sensor signals
GREEN  = I2C connections
BLUE   = GPIO outputs
WHITE  = Misc/Other

## COMPONENT CHECKLIST

Before connecting power, verify:

□ All sensor VCC → 5V rail (no direct to ESP32 5V pin)
□ All sensor GND → GND rail (no direct to ESP32 GND pin)
□ Each LED has 220Ω resistor in series
□ LED polarity correct (long leg = +)
□ No crossed wires or shorts
□ I2C connections (SDA=21, SCL=22)
□ Analog sensors on ADC1 pins only
□ USB cable connected

## VOLTAGE LEVELS

ESP32 Specifications:
  - Operating Voltage: 3.3V logic
  - Input Voltage (USB): 5V
  - GPIO Output: 3.3V
  - GPIO Input: 0-3.3V (5V tolerant on some pins)
  - Maximum Current per GPIO: 12mA
  - Maximum Total Current: 40mA

LED Current with 220Ω:
  - Red LED: ~8mA ✓
  - Green LED: ~8mA ✓
  - Blue LED: ~8mA ✓
  - Yellow LED: ~8mA ✓
  - Total: ~32mA ✓ (within safe limits)

## TROUBLESHOOTING QUICK REFERENCE

Problem: LED not lighting
  → Check: Polarity, resistor present, GPIO pin correct

Problem: Sensor reads 0 or max
  → Check: Power connections, signal wire, sensor itself

Problem: LCD shows nothing
  → Check: I2C address, contrast pot, power, SDA/SCL

Problem: WiFi won't connect
  → Check: SSID/password, 2.4GHz network, signal strength

Problem: ESP32 won't upload
  → Check: USB cable (data type), drivers, COM port selected
